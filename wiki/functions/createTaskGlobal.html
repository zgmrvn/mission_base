createTaskGlobal

Cette fonction permet de créer une tâche chez tous les joueurs, peu importe qu'elle soit exécutée côté serveur ou côté client. Le JIP est pris en charge.

Syntaxe

  • label - c'est ce label qu'il faudra passer pour modifier l'action
  • titre - le titre de la tâche qui apparaîtra dans la liste des tâches
  • description - description de la tâche lorsqu'elle est séléctionnée
  • notification - (facultatif) faut-il notifier les joueurs (valeur par défaut : false)

Exemples

mission/monScript.sqf
