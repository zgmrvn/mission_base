addActionGlobal

Description : Cette fonction permet d'ajouter une action chez tous les clients. Elle peut être utilisée depuis le serveur, sur un objet qui vient d'être créer. La problématique de JIP est prise en compte.

Syntaxe : [objet, titre, script, label, supprObjet, supprAction] call CRP_fnc_addActionGlobal

Paramètres :

  • objet : object - l'object sur lequel ajouter l'action
  • titre : string - le titre de l'action
  • script : string - le script à éxecuter (global). passer objNull si pas de script
  • label : string - référence de l'action, exemple : "action1", "ordinateur", "prendreTelephone"
  • supprObjet : bool - faut-il supprimer l'objet
  • supprAction : bool - faut-il supprimer l'action

exemple :

[
    _ordinateur,
    "Pirater ordinateur",
    "autoDestruction.sqf",
    "piratage",
    false,
    true
] call CRP_fnc_addActionGlobal;
