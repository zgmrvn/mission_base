addActionGlobal

Cette fonction permet de de gérer des actions de manière globale.

  • Object - l'oject sur lequel ajouter l'action
  • String - le titre de l'action
  • String - le script à éxecuter (global). objNull si pas de script
  • String - référence de l'action, exemple : "action1", "ordinateur", "prendreTelephone"
  • Bool - faut-il supprimer l'objet
  • Bool - faut-il supprimer l'action
