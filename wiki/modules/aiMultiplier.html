aiMultiplier

Ce module donne accès à un paramètre supplémentaire dans le lobby qui correspond à un multiplicateur d'IA et qui peut ensuite être utilisé par d'autres modules de création d'IA. Vous pouvez désactiver le module depuis le fichier de configuration du module modules/iaMultiplier/config.hpp. Si désactivé, la quantité d'IA des autres modules ne sera pas modifiée mais le paramètre apparaîtra quand même dans le lobby.
