intro

Afin de faciliter la gestion de l'intro, il est recommandé d'utiliser le fichier mission/intro/intro.sqf. Ce script fait parti d'une solution de gestion de l'intro qui permet de choisir depuis les paramètres de mission si l'intro doit être jouée. Cette solution permet également d'éviter de rejouer l'intro dans le cas d'une reconnexion à la mission ou encore de ne pas jouer l'intro lorsque vous êtes en édition. Vous pouvez désactiver l'intro depuis le fichier de configuration du module modules/intro/config.hpp.
